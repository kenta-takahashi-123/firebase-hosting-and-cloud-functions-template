# Template for Firebase Hosting and Firebase for Cloud Functions

## How to use

Simply copy and paste this repository files to your repository.

## Setup

```bash
yarn install
yarn run login
```

## Local development

以下で `http://localhost:5000` で、ブラウザで確認できるようになります。

```bash
yarn run serve
```

## Development deploy

```bash
yarn run deploy:dev
```

## Production deploy

```bash
yarn run deploy:prod
```
